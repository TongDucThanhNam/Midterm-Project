## Chạy chương trình

- Bước 1: Cài đặt các thư viện cần thiết

```
npm install
```

- Bước 2: Chạy chương trình

```
npm start
```

- Bước 3: WebNodeJS sẽ chạy trên đường link: http://localhost:8080/

Vào chức năng chat: http://localhost:8080/chat
